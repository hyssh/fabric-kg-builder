# Demo dataset — Microsoft Surface service guides

This demo ingests **publicly available Microsoft Surface service guides** (PDF). The
PDFs are **not** committed to this repository; download them yourself into a local folder before running ingestion.

## Where to download

All guides are free, public downloads from Microsoft:

- **Microsoft Download Center — Surface service guides:**
  https://www.microsoft.com/en-us/download/details.aspx?id=100440
- **Surface Service Guide Center (index, Microsoft Learn):**
  https://learn.microsoft.com/en-us/surface/surface-service-guide-center

Open the Download Center link, select the guides you want, and save the PDFs into
this folder. The guides are intended for skilled technicians; they describe
field-replaceable components and step-by-step replacement procedures — exactly the
hierarchical *part → sub-assembly → product* relationships this demo visualizes.
